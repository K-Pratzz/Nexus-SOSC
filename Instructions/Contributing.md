# Contributing Guidelines

Thank you for contributing!

## Steps

1. Fork the repository
2. Create branch: feature/your-feature-name
3. Make your changes
4. Commit clearly
5. Open Pull Request

## Rules

- Keep code clean
- Only HTML, CSS and JS are allowed
- Comment ONLY when necessary
- One feature per PR
- Be respectful

## All Contributions Are Welcome

Small contributions like fixing typos and UI improvements are also welcome!
