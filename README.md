# College Event Manager

An open-source web app to manage and display college events. (part of Nexus SOSC event)

---

Features

- Add events with a date
- View upcoming events
- Delete events
- Simple UI

---

## Getting Started

1. Clone the repository
git clone git@github.com:vidit401/Nexus-SOSC.git

2. Open `index.html` in browser
